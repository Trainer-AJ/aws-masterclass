This enables you to rapidly and reliably deliver features and updates
- AWS CodeBuild [classic build] is a fully managed build service that compiles source code, runs tests, and produces software packages that are ready to deploy. - AWS CodeDeploy [classic release] automates code deployments to any instance, including Amazon EC2 instances and on-premises servers.
